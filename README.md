# Описание программы

Голосовалка для команд выбора подходящих большинству настроек eslint.

## Установка проекта
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
